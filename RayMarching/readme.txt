Реализация алгоритма трассировки лучей OpenGL

Мной в данном проекте реализовано только передвижение камеры внутри main.cpp и фрагментный шейдер (за исключением функций main и EyeRayDir)
Используются функции расстояния до поверхностей, несколько типов материалов. Реализованы резкие тени, затемнение -Ambient Occlusion и затухающее по модели Фонга от двух источников света.

Сборка и запуск:
	mkdir build
	cd build
	cmake ..
	make
	./main

Рендеринг займет 5-10 секунд

Управление - WASD, Пробел и Z - для перемещения позиции камеры, мышь - для вращения сцены
0 - вернуться в начальное положение
1, 2 - демонстрация затемнения в углах с использованием алгоритма Ambient Occlusion
